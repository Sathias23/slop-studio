Read generation metadata (prompt, seed, model, etc.) from a ComfyUI-generated image.

$ARGUMENTS should be a file path to a PNG image generated by ComfyUI.

Steps:
1. Run the following Python script via `python3 -c` to extract the metadata, substituting the actual image path:

```
python3 -c "
import json, sys
from PIL import Image

img = Image.open(sys.argv[1])
info = img.info or {}
img.close()

if 'prompt' in info:
    prompt_data = json.loads(info['prompt'])
    print('=== PROMPT DATA ===')
    print(json.dumps(prompt_data, indent=2))
else:
    print('No prompt metadata found in image.')

if 'workflow' in info:
    workflow_data = json.loads(info['workflow'])
    print()
    print('=== WORKFLOW ===')
    print(json.dumps(workflow_data, indent=2))
" "$IMAGE_PATH"
```

2. From the prompt data, extract and present to the user:
   - **Prompt text**: Look for nodes with `class_type` "CLIPTextEncode" — the `inputs.text` field contains the prompt
   - **Seed**: Look for sampler nodes (e.g. `class_type` "KSampler" or "SamplerCustom") — the `inputs.seed` or `inputs.noise_seed` field
   - **Model**: Look for loader nodes (e.g. `class_type` "UnetLoaderGGUF" or "CheckpointLoaderSimple") — the `inputs.unet_name` or `inputs.ckpt_name` field
   - **Image dimensions**: Look for "EmptyLatentImage" or "EmptySD3LatentImage" nodes — `inputs.width` and `inputs.height`
   - **Steps / CFG / Sampler**: From sampler nodes — `inputs.steps`, `inputs.cfg`, `inputs.sampler_name`
   - **Guidance**: Look for "FluxGuidance" nodes — `inputs.guidance`

3. Present a clean summary to the user with the key parameters. If the workflow data is also present, mention that the full workflow is available if they want to see it.

Note: Only PNG images from ComfyUI contain this metadata. JPEG or WebP outputs will not have it.
